DJS-JPN とは？

DJS-JPN (Discord.JS Japan) は、主に Discord.JS を使った Discord ボットを開発をしているグループです。

現在は、６人のメンバーが参加しており、そのうちの３人が独自のボットを開発しています。

最近では、メンバー合同で、 Blade (旧名: Micro) という多機能ボットを開発しています。

こちらから DJS-JPN が開発したボットの確認ができますので、是非ご覧ください。

グループの目的

１．今の Discord には存在しない機能を搭載するボットを開発する。

DJS-JPN に加わりたい

DJS-JPN の開発メンバーになるには条件が必要です。

１．JavaScript、Node.js、Discord.js を理解している
２．Discord、Github のアカウントを持っている

上記の条件が完了していれば GitHub Organization への招待を送信します。
公式 Discord グループに参加（PR）

以下の「参加」ボタンをクリックし、「招待を受け取る」をクリックすることで、 DJS-JPN の公式 Discord サーバーに参加することができます。
